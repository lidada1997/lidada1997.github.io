### 📖 项目简介

基于多Agent协作的自动化数据分析系统，能够自动完成数据采集、清洗、分析、可视化和报告生成的完整流程。

### ✨ 核心特性

| 特性 | 说明 |
|------|------|
| **多Agent协作** | 5个专业Agent分工协作 |
| **多格式支持** | CSV, Excel, JSON, TXT, Parquet |
| **智能分析** | 描述性统计、相关性分析、趋势预测、异常检测 |
| **自动可视化** | 自动生成直方图、热力图、趋势图等 |
| **报告生成** | 自动撰写专业的Markdown格式报告 |
| **双版本实现** | Python原生 + Dify低代码工作流 |

### 🏗️ 系统架构

```
用户上传数据
    ↓
[DataCollectorAgent] → 数据采集与质量检查
    ↓
[DataCleanerAgent] → 数据清洗与预处理
    ↓
[StatisticalAnalyzerAgent] → 统计分析
    ↓
[VisualizationAgent] → 可视化生成
    ↓
[ReportGeneratorAgent] → 报告撰写
    ↓
输出：分析报告 + 可视化图表
```

### 📊 分析能力

- **描述性统计**：均值、中位数、标准差、分位数、峰度、偏度
- **相关性分析**：Pearson相关系数矩阵、强相关变量识别
- **趋势分析**：时间序列趋势检测、移动平均、线性回归
- **异常检测**：IQR方法、Z-score方法

### 🛠️ 技术栈

- Python 3.10+
- Pandas / NumPy - 数据处理
- SciPy - 统计分析
- Streamlit - Web界面
- Plotly / Matplotlib - 可视化
- Dify - AI工作流编排
